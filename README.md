## 🚀 Getting Started

To run these projects locally, it is highly recommended to use [Conda](https://docs.conda.io/en/latest/miniconda.html) to manage your Python environment and dependencies.

### Environment Setup (Conda)

**1. Create a new Conda environment:**
We use Python 3.9 as it offers excellent stability for OpenCV and FastAPI.
```bash
conda create --name cv_env python=3.9 -y
